# FotIQ  

FotIQ is a football quiz web application that tests user's trivia knowlege on football.  

## Features  
- User Signup & Sign-in with Firebase Authentication  
- Secure Authentication Handling  
- React-based Frontend
- Quiz

## Tech Stack  
- React.js  
- Firebase Authentication  
- CSS for Styling
